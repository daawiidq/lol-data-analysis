# League of Legends Data Analysis 🎮

## Overview
This project analyzes League of Legends match data to understand player performance and win rate trends.

## Features
- Calculated overall win rate
- Analyzed champion win rates
- Visualized performance trends using graphs

## Tech Stack
- Python
- Pandas
- Matplotlib

## What I Learned
- Data cleaning and preprocessing
- Basic data analysis techniques
- How to extract insights from gameplay data
